# NyxFoundation Research Interests

NyxFoundation メンバーのリサーチアイデア・興味トピックを蓄積する場所。
誰でも閲覧・コメント可能。issue の open / close は Nyx メンバーのみ。

## 目的

- 社内外の研究者・学生・エンジニアに向けて、Nyx が現在注力しているテーマを可視化する
- 個人の興味からチームの共同研究へつなげるための入口を作る
- 外部コミュニティとの接続点として活用する

## 参加方法

| あなたの役割 | できること |
|---|---|
| **Nyx メンバー** | Issue を作成・オープン・クローズ・編集 |
| **外部の研究者・学生・エンジニア** | Issue にコメント・反応・ディスカッション参加 |

外部の方も自由にコメント・フィードバックを歓迎します。
興味あるトピックがあれば気軽にコメントしてください。

## Issue テンプレート

新規に研究アイデアを登録する際は、以下のテンプレートに従ってください：

- 概要（1〜2文の elevator pitch）
- なぜ重要なのか（Why）
- この研究の主な貢献・期待される成果（What）
- 先行研究・関連ワーク
- Nyx の関連する取り組み・シナジー
- 対象となる読者・協力者
- 想定される難易度・期間
- キーワード・タグ
- その他の関連リンク
- 連絡先・ディスカッション
